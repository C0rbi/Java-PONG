# Java PONG
Un projet personnel pour m'entraîner en Java : une version simple du **Pong**, codée uniquement avec les bibliothèques standards de Java.

---

## Instalation / Jouer
```bash
git clone https://github.com/C0rbi/Java-PONG
```
### Avec IDE
Ouvre le projet dans ton IDE
et exécute directement la classe **`main.java`**

### Sans IDE
``` bash
cd Java-PONG
javac src/*.java
java -cp src main
```


## Controles
- Joueur 1 : Z / S
- Joueur 2 : ↑ / ↓
Fermer la fenètre pour quitter le jeu
